**Автор:** [Ваше Имя и Фамилия]

## Описание программы

Book Tracker - это десктопное приложение с графическим интерфейсом для управления списком прочитанных книг. Программа позволяет:

- Добавлять книги с указанием названия, автора, жанра и количества страниц
- Просматривать список всех добавленных книг в удобной таблице
- Фильтровать книги по жанру и количеству страниц
- Удалять книги из списка
- Просматривать статистику (общее количество книг, страниц, среднее значение)
- Автоматически сохранять данные в JSON файл

## Требования для запуска

- Python 3.7 или выше
- Стандартные библиотеки Python (tkinter, json, os, datetime)
